# Java Servlet Project

Web application developed using Java Servlets.

## Technologies
- Java
- Java Servlets
- NetBeans
- Apache Tomcat

## How to Run
1. Open the project in NetBeans
2. Configure Apache Tomcat
3. Run the project on the server

## Project Structure
- src/ → Java source code
- web/ → JSP files and web resources
